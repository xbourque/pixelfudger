ADJUSTING COMFYUI PREFERENCES

To make using ComfyUI easier, we need to adjust a few preferences.
This document will guide you to adjust these preferences.

ADJUSTING COMFYUI MAIN SETTINGS

OPEN COMFYUI SETTINGS

Toolbar -> Gear icon.

Link Render Mode -> Straight.

Widget Value Control Mode -> Before.

ADJUSTING COMFYUI CUSTOM NODES SETTINGS

VHS Advanced Previews -> ON.

This setting will only appear if the ComfyUI-VideoHelperSuite custom nodes are installed.

Anything Everywhere animate UE links -> Off.

This setting will only appear if the Use Everywhere (UE Nodes) custom nodes are installed.

Anything Everywhere show links -> Selected Nodes.

This setting will only appear if the Use Everywhere (UE Nodes) custom nodes are installed.

CLOSE COMFYUI SETTINGS

Click X.

ADJUSTING COMFYUI 'MANAGER' SETTINGS

OPEN THE 'MANAGER'

Toolbar -> Manager.

Preview Method -> Auto.

Badge -> #ID Nickname.

Default UI -> Queue.

CLOSE THE 'MANAGER'

Click Close.
