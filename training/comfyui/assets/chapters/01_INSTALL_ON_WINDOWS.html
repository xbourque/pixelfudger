INSTALLING COMFYUI ON WINDOWS

INSTALL 7-ZIP

DOWNLOAD 7-ZIP

Go to 7-Zip.org.
Download 7-Zip for Windows 64-bits x64.

RUN THE 7-ZIP INSTALLER

Locate the download and run the 7-Zip installer.

CREATE A DESTINATION DIRECTORY

CREATE AN EMPTY COMFYUI DIRECTORY

In a File Explorer, create a New Folder at C:\comfyui\

Plan for enough disk space! ComfyUI (including AI models required for this tutorial) uses about 70GB of disk space.

If you don't want to install on your C:\ drive, choose an install directory close to the root folder of your desired disk (for example: D:\comfyui\). Very long file paths can break Python on Windows.

INSTALL COMFYUI

DOWNLOAD COMFYUI

In a web browser, go to the ComfyUI GitHub Release Page.
Click on Download Link with stable pytorch 2.3 cu121.
Save the download to C:\comfyui\.

EXTRACT THE COMFYUI ARCHIVE

In a File Explorer, locate the ComfyUI .7z file.
Right-click on the file.
Choose Show More Options... -> 7-Zip -> Extract Here

The ComfyUI archive (inlcuding a self-contained Python 3.11 environment) is extracted to C:\comfyui\ComfyUI_windows_portable_nvidia_cu121_or_cpu\.

After extracting the contents of the archive, you can move the ComfyUI .7z file to the Recycle Bin if you wish.

RENAME THE FOLDER

Rename the ComfyUI_windows_portable_nvidia_cu121_or_cpu folder to portable.

The ComfyUI self-contained installation is now located at C:\comfyui\portable\.

INSTALL THE 'MANAGER'

DOWNLOAD THE COMFYUI MANAGER

In a Web Browser, go to the
ComfyUI-Manager GitHub repository.
Click Code (top right). Choose Download ZIP.

EXTRACT THE COMFYUI-MANAGER ARCHIVE

In a File Explorer,
right-click on ComfyUI-Manager-main.zip.
Choose Show More Options -> 7Zip -> Extract Here

MOVE COMFYUI-MANAGER TO 'CUSTOM NODES'

In a second File Explorer. Navigate to C:\comfyui\portable\ComfyUI\custom_nodes.
Move the ComfyUI-Manager-main folder to the custom_nodes folder.

START THE COMFY UI SERVER

START THE COMFYUI SERVER

In a File Explorer, navigate to C:\comfyui\portable\.
Double-click on run_nvidia_gpu.bat.

If you do not have a supported GPU, you can run ComfyUI in (slow!) CPU mode with run_cpu.bat instead.

The ComfyUI server starts in a new Command Prompt window.

A new web browser window opens and navigates automatically to http://127.0.0.1:8188.
Make sure the Manager button is visible in the toolbar on the right.
Bookmark this address for easy access in the future.

To shutdown the ComfyUI server, simply close the Command Prompt window.
(Don't shut it down yet, we still have a few things to configure!)
